# JD Optimization

## When It Runs

JD optimization appears only when a job description is provided.

## Output

- Must-have skills
- Preferred skills
- Responsibilities
- Missing skills
- Experience gaps
- Optimization suggestions

## Relationship To ATS Missing Skills

The ATS engine extracts JD target terms with deterministic rules. The AI layer can add richer interpretation of gaps and responsibilities, but it does not overwrite the ATS score.
