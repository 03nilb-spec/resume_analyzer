# UI/UX Flow

## Flow

1. Upload resume.
2. Paste optional job description.
3. Click Analyze Resume.
4. Review ATS score and AI coaching dashboard.

## Dashboard

- ATS Score
- Score Breakdown
- AI Career Coach Summary
- Role Fit Analysis
- Top 3 Priority Fixes
- Resume Strengths
- Rewrite Suggestions
- JD Optimization
- ATS Matched and Missing Skills
- Formatting Issues

## Error States

- Unsupported file type.
- Oversized file.
- Empty or scanned resume.
- AI insights unavailable while ATS analysis still succeeds.
