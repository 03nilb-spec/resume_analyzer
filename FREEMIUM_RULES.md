# Freemium Rules

## Public Users

Public users can upload a resume and receive basic ATS analysis. AI Career Coach is gated behind Google sign-in.

## Free Signed-In Users

Free signed-in users receive:

- 3 AI analyses per month
- Basic ATS score
- Skill count summary
- Saved analysis history

Free users do not see full matched/missing skill lists in the UI. The backend still returns those arrays so Premium can unlock them later without changing the analysis API.

## Premium Placeholder

Premium is not implemented yet. Premium messaging points users to `/pricing`, where checkout is intentionally disabled.

Planned Premium benefits:

- Unlimited AI analyses
- Full matched/missing skills
- AI Resume Rewriter
- AI resume improvement
- Job-specific optimization
- Downloadable reports
